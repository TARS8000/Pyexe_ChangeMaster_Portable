# Pyexe_ChangeMaster_portable
## 概要
Pythonファイルを簡単にexe化します。

## 対応環境
windows　64bit

動作にはPython本体が必要です。初回動作時はsetup.batを押せば簡単に環境構築ができます。
## 制作環境
Spyder

Intellij IDE

Gemini Pro
## ロジック
①Pyファイルが読み込まれる。

②Pyの文から必要モジュールを検出する。

③依存フォルダを検出する。

④フォルダ出力場所を参照してもらう。

⑤オプションを選択してもらう。

⑥変換ボタンが押される

⑦変換環境を参照（ポータブルPy環境があればそれを参照、なければシステムPy環境を参照）

⑧出力先にフォルダを生成。

⑨対象環境にモジュールがあるか探す。なければpipでインストール。

⑩必要モジュールをすべて含めさせオプションもつけてpyinstallerで変換。

⑪出力フォルダ内を整理。

⑫バージョン情報やZoneIDを削除しSAC対策。

⑬完了。
## 注意
・SACに弾かれるケースが多いです。windows設定から変えてください。
